Import Data¶

This page lists all of the available data import routines through the Katana Local API. Import routines return a new Graph object.

The import_data module provides a several graph conversion routines which take external data and convert it into a Katana Graph object.

katana.local.import_data.from_adjacency_matrix(adjacency: numpy.ndarray, property_name: str = 'weight') → katana.local_native.Graph[source]¶

Convert an adjacency matrix with shape (n_nodes, n_nodes) into a Graph with the non-zero values as an edge weight property.

This preserves node IDs, but not edge IDs.

katana.local.import_data.from_csr(arg0: numpy.ndarray[numpy.uint64], arg1: numpy.ndarray[numpy.uint32]) → std::shared_ptr<katana::PropertyGraph>¶

Create a new Graph from a raw Compressed Sparse Row representation.

Parameters

  • edge_indices (numpy.ndarray or another type supporting the buffer protocol. Element type must be an integer.) – The indicies of the first edge for each node in the destinations vector.

  • edge_destinations (numpy.ndarray or another type supporting the buffer protocol. Element type must be an integer.) – The destinations of edges in the new graph.

Returns

the new Graph

katana.local.import_data.from_edge_list_arrays(sources: numpy.ndarray, destinations: numpy.ndarray, property_dict: Optional[Dict[str, numpy.ndarray]] = None, *, node_types: Optional[katana.local.entity_type_array.EntityTypeArray] = None, edge_types: Optional[katana.local.entity_type_array.EntityTypeArray] = None, **properties: numpy.ndarray) → katana.local_native.Graph[source]¶

Convert an edge list represented as two parallel arrays into a Graph.

This preserves node IDs, but not edge IDs.

katana.local.import_data.from_edge_list_dataframe(df, source_column: Union[str, int] = 'source', destination_column: Union[str, int] = 'destination', property_columns: Optional[Collection[Union[str, int]]] = None, *, node_types: Optional[katana.local.entity_type_array.EntityTypeArray] = None, edge_types: Optional[katana.local.entity_type_array.EntityTypeArray] = None)[source]¶

Convert an edge list in the form of a dataframe-like object into a Graph with edge properties.

Parameters

  • df (A data frame like type, such as a pandas.DataFrame or even a dict of numpy arrays.) – The edge list and edge properties.

  • source_column – The name or ID of the column which holds edge sources.

  • destination_column – The name or ID of the column which holds edge destinations.

  • property_columns – The names or IDs of columns which should be used as edge properties. If this is None, use all columns other than sources and destinations specified above.

katana.local.import_data.from_edge_list_matrix(edges: numpy.ndarray) → katana.local_native.Graph[source]¶

Convert an edge list with shape (n_edges, 2) into a Graph.

This preserves node IDs, but not edge IDs.

katana.local.import_data.from_graphml(path: Union[str, pathlib.Path], chunk_size: int = 25000, *, txn_ctx=None)[source]¶

Load a GraphML file into Katana form.

Parameters

  • path (Union[str, Path]) – Path to source GraphML file.

  • chunk_size (int) – Chunk size for in memory representations during conversion. Generally this value can be ignored, but it can be decreased to reduce memory usage when converting large inputs.

  • txn_ctx – The tranaction context for passing read write sets.

Returns

the new Graph
