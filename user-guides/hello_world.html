First steps in Katana on One HostÂ¶

Katana is easiest to use in Conda (for Python users). So make sure you have a Conda installation. The Katana open-source packages only support Linux x86-64. You can create a Conda environment with Katana and Jupyter installed with:

conda env create -f https://raw.githubusercontent.com/KatanaGraph/katana/master/katana.yml

Finally, you can start Jupyter with the following and open the notebook normally:

conda run -n katana jupyter lab
