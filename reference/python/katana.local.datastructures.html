Data Structures¶

This module contains data structures designed for use with Galois’ parallel loops. They are instantiated for several element types, but cannot support custom element types.

The instantiations are:

  • np.uint64 (with C element type uint64_t)

  • int (with C element type int64_t)

  • np.uint32 (with C element type uint32_t)

  • np.int32 (with C element type int32_t)

  • float (with C element type double)

  • np.float32 (with C element type float)

The instantiation is selected by indexing the type, for example InsertBag[int] or NUMAArray[np.uint64].
